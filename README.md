# Student Grade Calculator

## Project Overview
This represents a Python program that calculates a student's grade based on their marks (0-100). It includes input validation and encouraging messages.

## Features
- Accepts student name and marks.
- Validates that marks are between 0 and 100.
- Returns a letter grade (A-F).
- Provides a custom encouraging message.

## How to Run
1. Make sure Python is installed.
2. Run command: `python grade_calculator.py`
